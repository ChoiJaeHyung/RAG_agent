# 외부 지식 판단 방식별 비용 분석

## 📊 현재 시스템 LLM 호출 패턴 분석

### 1회 검색당 LLM 호출

#### 기본 호출 (필수)
1. **Agent Decision (ReAct 루프)**: 1-5회
   - 각 iteration마다 도구 선택
   - 평균 3회 호출 가정

2. **외부 지식 획득** (조건부): 0-1회
   - 기술 용어 설명 생성
   - 현재: 약 30-40% 케이스에서 호출

3. **최종 답변 생성**: 1회
   - 항상 실행

**현재 시스템 평균:**
- Agent Decision: 3회
- 외부 지식: 0.35회 (35% 확률)
- 답변 생성: 1회
- **총 평균: 4.35회 / 검색**

## 📝 각 LLM 호출별 토큰 사용량 추정

### 1. Agent Decision (도구 선택)
```python
# System prompt: ~1,500 tokens
# User message: ~300 tokens (질문 + 상태)
# Tool definitions: ~2,000 tokens
# Response: ~200 tokens

Input:  3,800 tokens
Output:   200 tokens
Cost:   $0.00057 + $0.00012 = $0.00069 / 호출
```

### 2. 외부 지식 획득
```python
# System prompt: ~100 tokens
# User message: ~150 tokens (용어 리스트)
# Response: ~300 tokens (설명)

Input:    250 tokens
Output:   300 tokens
Cost:   $0.0000375 + $0.00018 = $0.0002175 / 호출
```

### 3. 최종 답변 생성
```python
# System prompt: ~500 tokens
# User message: ~8,500 tokens (질문 + 문서 컨텍스트 + 외부지식)
# Response: ~1,500 tokens

Input:  9,000 tokens
Output: 1,500 tokens
Cost:   $0.00135 + $0.0009 = $0.00225 / 호출
```

### 4. 문서 충분성 판단 (신규 추가)
```python
# System prompt: ~150 tokens
# User message: ~1,200 tokens (질문 + 문서 요약)
# Response: ~300 tokens (JSON 판단)

Input:  1,350 tokens
Output:   300 tokens
Cost:   $0.0002025 + $0.00018 = $0.0003825 / 호출
```

**비용 증가:**
